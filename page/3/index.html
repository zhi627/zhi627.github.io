python 变量与对象

发表于 2017-06-01 | 分类于 python | 阅读次数

简介

  1. Python中一切都是对象，变量总是存放的是对象的引用。这是一个普遍的法则。
  2. 变量无类型，对象有类型。
  3. 只要用一个变量，就要给这个变量赋值。
  4. 当一个对象没有变量指向它的时候，它便进入了垃圾收集过程。
  5. Python的“动态类型”机制，负责检查变量的对象引用适用操作。如果该对象不适用该操作，则会直接报错。
