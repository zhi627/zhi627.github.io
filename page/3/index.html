python 变量与对象

发表于 2017-06-01 | 分类于 python | 阅读次数

简介

  1. Python中一切都是对象，变量总是存放的是对象的引用。这是一个普遍的法则。
  2. 变量无类型，对象有类型。
  3. 只要用一个变量，就要给这个变量赋值。
  4. 当一个对象没有变量指向它的时候，它便进入了垃圾收集过程。
  5. Python的“动态类型”机制，负责检查变量的对象引用适用操作。如果该对象不适用该操作，则会直接报错。

zabbix低级别发现

发表于 2017-05-25 | 分类于 zabbix | 阅读次数

简介

zabbix低级别发现(LLD)在单机多实例的情况下比较有用,很多情况使用netstat -nlp获取绑定ip以及端口的服务程序。
例如:MySQL 、Redis 等常用server端程序。或者直接获取服务端口只检测端口是否可到达的时候十分方便。以及设置相关监控项、触发器、图表等,
省去人工添加模板的步骤。当然还是要创建相关LLD的模板
主要注意几点:

  1. 是否开启了ipv6地址允许,可见为什么 netstat 对某些服务只显示了 tcp6 监听端口,是否需要禁用ipv6,根据情况处理LLD脚本
  2. 是使用UDP还是TCP协议的服务程序
  3. 使用netstat的权限问题,默认用户是zabbix,运行权限需要是root,设置sudo规则
  4. zabbix是nologin用户需要注释sudo配置文件中的Defaults requiretty
