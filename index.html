Hexo与GiHub Pages 搭建个人博客

本文将详细介绍如何在 Centos 系统使用 Hexo 以及GitHub Pages 搭建个人博客

注册GitHub账号

安装Git

安装Node.js以及npm

在国内用户可以更新yum源以及npm源为国内的，这样下载安装更快，可参考 yum源更换 以及 npm源更换

安装Hexo

用Hexo创建blog

测试Hexo是否可以运行

  • 安装hexo-server插件，可将服务器独立成个别模块
  • 启动服务器，提示博客会在http://localhost:4000/下启动。 在服务器启动期间，Hexo会监视文件变动并自动更新，无须重启服务器。不断刷新网页即可看到更新。

创建ssh私钥与公钥

调用ssh-keygen生成对应用户的私钥与公钥文件id_rsa与id_rsa.pub

  • 在自己创建的GitHub Pages项目Settings找到Deploy keys,并复制id_rsa.pub公钥内容到Key文本框中
  • 勾选Allow write access复选框,设置可以通过密钥方式修改博客内容

设置git账号

  • 设置为自己GitHub账号以及邮箱

设置hexo提交更新到GitHub

  • 设置hexo创建的博客文件夹的_config.yml中deploy设置为git部署
    deploy:

    type: git
repo: git@github.com:account/blogname.github.io.git
branch: master

  • 安装hexo-deployer-git插件可以使用git进行部署

  • 部署网站并且提交到GitHub
