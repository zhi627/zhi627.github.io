shell 数组详解

发表于 2017-08-24 | 分类于 shell | 阅读次数

简介

转自 Bash Array 数组教程
数组是包含多个相同或不同数据类型的集合，数组索引从零开始。

样例讲解使用

声明数组，赋值

当有变量为如下格式的时候，Bash 会自动创建数组。

  • name为数组名称
  • index为任意数字，或表达式的最终计算值大于等于零。可以显式声明数组通过 declare -a arrayname

访问数组元素使用括号，如 ${name[index]}

声明时初始化数组

通过指定元素列表来声明数组，就不用一个一个分别初始化数组元素的，用空格隔开在括号里。

如果数组元素包含空格符，用引号括起来。

declare -a 声明一个数组以及括号中的所有元素为数组元素。

输出整个 Bash 数组

有多种不同的方法来输出整个数组。如果索引是 @ 或者 *，则引用数组所有元素。也可以使用循环，遍历数组中每个元素再输出。

如果引用数组元素，而不提供索引的话，就是引用数组的第一个元素，即索引为零的元素。

Bash 数组长度

可以使用特殊参数 $# 来获得数组长度。

数组第 n 个元素的长度

${#arrayname[n]} 为数组第 n 个元素的长度。
