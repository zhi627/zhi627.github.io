创建并导入数据库文件

  • 创建zabbix数据库,mysql5.7默认需要设置更为复杂的秘密

  • 导入数据库文件,记住zabbix-server-mysql-3.0.X,是对应自己安装zabbix的版本

设置虚拟主机

  • 根据httpd版本，可能会有不同语法，根据情况修改配置文件，常见是httpd2.2与httpd2.4的区别
    Alias /zabbix /usr/share/zabbix
<Directory "/usr/share/zabbix">
    Options FollowSymLinks
    AllowOverride None
    Allow from all
    <IfModule mod_php5.c>
        php_value max_execution_time 300
        php_value memory_limit 128M
        php_value post_max_size 16M
        php_value upload_max_filesize 2M
        php_value max_input_time 300
        php_value always_populate_raw_post_data -1
        php_value date.timezone PRC
     </IfModule>
</Directory>

配置数据库以及登录

  • 打开浏览器,访问http://hostname/zabbix/setup.php, 填写mysql相关信息,host填写为127.0.0.1以及检测软件是否满足要求
  • 编辑/etc/zabbix/zabbix_server.conf文件设置mysql数据库信息
  • service zabbix-server start
  • 登录，默认账号:Admin,密码:zabbix

优化以及配置

  • 根据监控情况进行相关配置，如果监控主机数较多，需要配置mysql的max-connections选项
  • 监控方式zabbix_agentd主动模式多的话,需要配置Trapper数量
  • 监控方式zabbix_agentd被动方式多的话,需要配置Poller数量
  • 当监控主机数量较多,建议使用主动模式或者proxy模式
