zabbix创建监控流程

发表于 2017-05-22 | 分类于 zabbix | 阅读次数

通过zabbix监控主机,一般方式以及流程步骤：

方式

  1. 先创建模板,然后模板应用到主机上
  2. 选择主机,在主机创建监控项、触发器等

流程

1.创建监控模板(可有可无)-> 2.创建应用集(可有可无) -> 3.创建监控项 -> 4.创建触发器 -> 5.创建动作(特指事件通知)

说明

名称 描述
模板 创建监控模板，定义相关应用集、监控项、触发器、图形、聚合图像、web监控
应用集 一组相关有联系的监控项
监控项 定义监控的数据以及监控(类型)方式
触发器 根据监控项返回值的情况判断其严重性以及标记问题信息
事件 事件基本无处不在,由触发器产生的其实都可以叫事件,事件包括正常以及异常的情况，并不是只有异常的情况
动作 可创建触发器的动作,或者自动发现、内部发现、自动注册的动作,定义在其发生时会如何处理,可以远程、本地调用或者发送消息

执行流程

zabbix-server端获取到监控项,如果有触发器将根据触发器的表达式产生事件,同时根据动作的配置,去执行动作的操作
